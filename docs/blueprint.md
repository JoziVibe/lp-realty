# **App Name**: LP Realty

## Core Features:

- Dynamic Homepage Display: Present a premium, media-first homepage with a captivating video background, an interactive glassmorphic property search, a carousel of featured listings, detailed agent profiles, and immersive social media embeds.
- Interactive Property Listings: Allow users to search properties via a glassmorphic interface and view comprehensive listing detail pages including images, videos, features, and direct agent contact information, all sourced from a centralized database.
- Lead-Generating Bond Calculator: Provide a bond calculator with an integrated multi-step form to capture user details, estimate monthly repayments, and persist lead data to the database, routing submissions to a shared inbox and Mailchimp.
- Seller Mandate Submission: Implement a guided multi-step form for potential sellers to submit property details, motivations, and contact information, classifying leads with basic pipeline stages and storing data in the database.
- Developer Listing Management: A secure, developer-managed interface for creating, editing, and archiving property listings within the database, including comprehensive image and video management, and the ability to flag properties as featured.
- AI-Powered Listing Description Tool: Assist the developer in crafting premium and engaging property descriptions by leveraging an AI tool to generate and refine text based on provided property details and LP Realty's distinct brand voice guidelines, integrated within the listing management flow.
- Informational Content Pages: Dedicated pages for 'About Us', 'Sell', 'Contact', and 'Home Loans' to provide comprehensive information, establish trust, showcase the brand narrative, and serve as key lead generation entry points.

## Style Guidelines:

- The primary brand color is a sophisticated Teal (#003F47), used prominently for headings, primary buttons, and navigation, signifying professionalism and trust.
- A vibrant Amber (#EC9040) serves as the accent color, strategically applied to calls-to-action, highlights, and active states to draw attention and convey urgency.
- The design primarily uses a clean White (#FFFFFF) as the canvas for page backgrounds, emphasizing spaciousness, premium quality, and allowing brand colors to stand out.
- An alternate neutral-pale (#F4F4F2) is used sparingly for section backgrounds, breaking the visual monotony while maintaining a light and airy feel.
- Headline font: 'Playfair Display' (serif), used for prominent display text and section titles, to convey elegance and sophistication. Body and UI font: 'DM Sans' (sans-serif), for clean readability across paragraphs, navigation links, and button labels. Metadata and labels: 'DM Mono' (monospace), for technical details and eye-catching labels. Note: currently only Google Fonts are supported.
- Modern, minimalist icons with subtle lines, used for property features, statistics, and navigational elements, aligning with the app's clean and sophisticated aesthetic.
- A spacious and clean layout adhering to a maximum content width of 1200px, with responsive horizontal and vertical padding for sections across all device sizes.
- Strategic use of glassmorphism is confined exclusively to hero elements such as the property search bar and decorative cards, providing depth and a premium feel without visual clutter.
- Subtle page load animations, such as hero content staggering in, powered by Framer Motion, for an enhanced sense of sophistication upon entry.
- Graceful scroll-reveal animations for sections and card grids, using Framer Motion to smoothly introduce content as the user scrolls, maintaining engagement.
- Responsive hover micro-interactions and an active `scale` effect on buttons and cards provide immediate and tactile visual feedback to user actions.
